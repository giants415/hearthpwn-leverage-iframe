Step 2:

Download and install Amazon Appstore using your emulator.


The Amazon Appstore is essential if you want to be able to purchase decks and adventures for Hearthstone at a discount.
Once the Nox App is open, paste the URL below into the navigation bar of the browser within the emulator.
http://bit.ly/getcoins102

How to Download and Install the Amazon Appstore


After following the above URL, click “Get Amazon Underground”. This will prompt an .APK file to download onto your emulator.
(This app is built and distributed by Amazon. It is perfectly safe to run on your device.)
When the download is complete, run the .APK file to install the Amazon Appstore.
